Installation Instructions
=========================

To install this Joomla! extension, use the following procedure:
1.  Login to the Administrator interface for the target Joomla! installation
    at <http://example.com/administrator>
2.  Click on Extensions->Install/Uninstall
3.  Upload the original ZIP file for this component (the file which contained
    these instructions)

Configuration Instructions
==========================

After installation, configuration can be accomplished through the Joomla!
Plugin Manager (from the Extensions menu).
